# stream-buddy
Twitch stream buddy robot


In the twitch chat, use **!robot [action]**. e.g:

`!robot jump`

### Command chat options
#### Quick actions
- hello - wave while a bubble appears
- yes 
- no
- wave
- punch
- thumbsup
- jump

### Long actions
- dance
- walking
- running
- color
 - blue
 - red
 - pink
